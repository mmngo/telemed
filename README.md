# HI 191 - Telemed (A Telemedicine Application)

### Install Pipenv
  
```
pip3 install pipenv
```


### Run project
1. Activate pipenv
```
pipenv shell
```

2. Install dependencies(SKIP if no new installed packages).
```
pipenv install -r ./requirements.txt
```

3. Run the server
```
python manage.py runserver
```
